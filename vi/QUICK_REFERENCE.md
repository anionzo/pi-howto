# Quick Reference

## Cài đặt

```bash
npm install -g @mariozechner/pi-coding-agent
```

## Bắt đầu

```bash
pi
/login
```

## Commands hay dùng

- `/model`
- `/settings`
- `/resume`
- `/new`
- `/session`
- `/tree`
- `/fork`
- `/compact`
- `/reload`
- `/hotkeys`

## CLI hay dùng

```bash
pi -c
pi -r
pi --no-session
pi --session <path>
pi --fork <path>
pi --model openai/gpt-4o
pi -p "Summarize this"
```

## Shortcut hay dùng

- `Ctrl+L` - chọn model
- `Ctrl+P` - cycle model
- `Shift+Tab` - cycle thinking
- `Ctrl+T` - toggle thinking
- `Ctrl+O` - toggle tool output
- `Ctrl+G` - external editor

## File locations

- `~/.pi/agent/settings.json`
- `~/.pi/agent/auth.json`
- `~/.pi/agent/sessions/`
- `~/.pi/agent/extensions/`
- `~/.pi/agent/skills/`
- `~/.pi/agent/themes/`
